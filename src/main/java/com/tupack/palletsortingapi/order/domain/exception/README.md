# Order Domain Exceptions

Este directorio está reservado para excepciones específicas del dominio de órdenes.

## Uso Futuro

Las excepciones de dominio puro (reglas de negocio) deberían colocarse aquí:
- Excepciones de validación de reglas de negocio del dominio Order
- Excepciones de invariantes de dominio
- Excepciones de políticas de negocio

## Excepciones Actuales

Actualmente las excepciones relacionadas con órdenes están en:
`common/exception/` - Para mantener compatibilidad con el código existente

Considerar mover en futuras refactorizaciones de arquitectura.
